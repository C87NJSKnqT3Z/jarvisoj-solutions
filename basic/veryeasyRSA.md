## 题解

exgcd 求个逆元

```python
def exgcd(a, b):
    if not b:
        return a, 1, 0
    d, y, x = exgcd(b, a % b)
    y -= a // b * x
    return d, x, y

def get_inverse(a, p):
    d, x, y = exgcd(a, p)
    return (x + p) % p
```
